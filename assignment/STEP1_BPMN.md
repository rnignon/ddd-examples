## 1. 전체 업무 흐름
```
[고객] → (대출 신청)
      → [대출 신청 관리 시스템]
            → (신청서 작성)
            → (서류 업로드)
            → (신청 정보 검증)
      
      → [사전 심사 시스템]
            → (신용 정보 조회) → [외부: NICE/KCB]
            → (AI 자동 심사)
            → (기본 자격 요건 검증)
            ├─▶ 탈락 → [고객] ← (탈락 통보) → 프로세스 종료
            └─▶ 통과 
      
      → [본 심사 시스템 (시스템이 심사 대상 건을 "심사 대기큐"에 자동 등록)]
            → (심사역 배정) → [심사팀장]
            → (상세 심사 수행) → [심사역]
                  → (소득 확인) → [외부: 국세청]
                  → (담보 평가) → [외부: 등기소]
                  → (재무 상태 분석)
                  → (상환 능력 평가)
            → (심사 의견 작성)
            → (필요시 추가 서류 요청(고객에게 알림 발송)
            → (시스템에 심사 완료처리 및 "승인 대기 상태"로 전환)
      
      → [승인 관리 시스템]
            → (승인 권한 판단)
            ├─▶ 3천만원 이하 → (AI 자동 승인)
            └─▶ 3천만원 초과 → (승인권자 검토) → [승인권자]
            → (최종 승인/조건부승인/거절 결정)
            → (고객에게 승인 결과 통보 (SMS, 앱 푸시, 이메일))
      
      → [대출 실행 시스템]
            → (약정 안내 발송) → [고객]
            → (모바일에서 약정 내용 확인) → [고객]
            → (전자 약정 체결) → [고객]
            → (계좌 실명 확인) → [외부: 금융결제원]
            → (자금 이체 실행)
            → (대출 원장 생성) → [코어뱅킹 시스템]
            → (실행 완료 통보) → [고객]
      
      → [상환 관리 시스템]
            → (매월 상환일 3일 전 알림 발송)
            → (자동 이체를 통한 정기 상환 처리)
            → (연체 모니터링)
            → (중도 상환 처리)
```

## 2. 세부 단계별 프로세스

### 2.1 대출 신청 단계
```
[고객]
   │
   ├─▶ (모바일 앱/웹 접속)
   │
   ├─▶ (본인 인증)
   │     ├─ 공동인증서
   │     └─ 간편인증
   │
   ├─▶ (상품 선택 및 한도 조회)
   │     └─▶ [대출 상품 관리] → (한도 계산)
   │
   ├─▶ (신청서 작성)
   │     ├─ 개인정보 입력
   │     ├─ 소득정보 입력
   │     ├─ 대출 금액/기간 입력
   │     └─ 대출 목적 입력
   │
   ├─▶ (필수 서류 업로드)
   │     ├─ 신분증 촬영
   │     ├─ 소득 증빙 서류
   │     └─ 재직 증명서
   │     └─▶ [OCR 시스템] → (자동 인식 및 검증)
   │
   ├─▶ (동의 처리)
   │     ├─ 개인정보 수집 동의
   │     └─ 신용정보 조회 동의
   │
   └─▶ (신청 제출)
         └─▶ 접수번호 발급
               └─▶ [고객] ← SMS/앱 푸시 알림
```

### 2.2 사전 심사 단계
```
[대출 신청 관리 시스템]
   │
   └─▶ (신청 접수 감지) → [사전 심사 시스템]
         │
         ├─▶ (서류 자동 검증)
         │     └─▶ [OCR 엔진] → 서류 진위 확인
         │
         ├─▶ (외부 신용 정보 조회)
         │     ├─▶ [NICE 평가 API]
         │     └─▶ [KCB API]
         │     └─▶ 신용등급, 연체 이력 수신
         │
         ├─▶ (AI 자동 심사)
         │     └─▶ [AI 심사 엔진]
         │           ├─ 연령 검증 (19-65세)
         │           ├─ 신용등급 검증 (6등급 이상)
         │           ├─ 소득 검증 (연 2,000만원 이상)
         │           ├─ 연체 이력 검증 (90일 이상 연체 없음)
         │           └─ DSR 계산 (40% 이하)
         │
         └─▶ (사전 심사 결과 판정)
               ├─▶ 통과 → 본심사 대기 상태로 전환
               │         └─▶ [고객] ← "사전심사 통과" 알림
               │
               └─▶ 탈락 → 신청 종료
                         └─▶ [고객] ← "심사 탈락" 알림 + 사유
```

### 2.3 본 심사 단계
```
[사전 심사 시스템]
   │
   └─▶ (본심사 대기큐 등록) → [본 심사 시스템]
         │
         ├─▶ (심사역 배정)
         │     ├─ [심사팀장] → 수동 배정
         │     └─ 또는 자동 배정
         │
         └─▶ [담당 심사역]
               │
               ├─▶ (신청 정보 통합 조회)
               │     ├─ 고객 기본 정보
               │     ├─ 신용 정보
               │     ├─ 업로드 서류
               │     └─ 사전 심사 결과
               │
               ├─▶ (소득 서류 진위 확인)
               │     └─▶ [국세청 API] → 소득 증빙 조회
               │
               ├─▶ (담보 평가) ※담보대출인 경우
               │     └─▶ [등기소 API] → 등기부 조회
               │           └─ LTV 계산 (60% 이내)
               │
               ├─▶ (재무 상태 분석)
               │     ├─ 기존 대출 현황 확인
               │     ├─ 신용카드 사용 패턴 분석
               │     └─ 상환 능력 평가
               │
               ├─▶ (추가 서류 요청) ※필요시
               │     └─▶ [고객] ← 추가 서류 요청 알림
               │           └─▶ [고객] → 서류 제출
               │
               └─▶ (심사 의견 작성 및 완료)
                     ├─ 심사 의견 입력
                     ├─ 추천 금액/금리 입력
                     └─ 승인 추천 또는 거절 의견
                     └─▶ 심사 완료 처리
```

### 2.4 승인 단계
```
[본 심사 시스템]
   │
   └─▶ (심사 완료) → [승인 관리 시스템]
         │
         ├─▶ (승인 권한 자동 판단)
         │     │
         │     ├─▶ 3천만원 이하 + 신용등급 3등급 이상
         │     │     └─▶ [AI 자동 승인]
         │     │           └─ 즉시 승인 처리
         │     │
         │     ├─▶ 3천-5천만원
         │     │     └─▶ [심사역 (대리~과장)]
         │     │           └─ 2시간 이내 처리
         │     │
         │     ├─▶ 5천-1억원
         │     │     └─▶ [심사팀장 (차장)]
         │     │           └─ 4시간 이내 처리
         │     │
         │     ├─▶ 1억-3억원
         │     │     └─▶ [심사부서장 (부장)]
         │     │           └─ 1영업일 이내 처리
         │     │
         │     └─▶ 3억원 초과
         │           └─▶ [여신위원회 (본부장, 임원)]
         │                 └─ 3영업일 이내 처리
         │
         └─▶ [승인권자]
               │
               ├─▶ (심사 내역 검토)
               │     ├─ 심사역 의견 확인
               │     ├─ 신용 정보 재확인
               │     └─ 리스크 평가
               │
               └─▶ (최종 결정)
                     ├─▶ 승인 → 실행 대기 상태로 전환
                     │         └─▶ [고객] ← "대출 승인" 알림
                     │
                     ├─▶ 조건부 승인 → 조건 충족 확인 후 승인
                     │         └─▶ [고객] ← "조건부 승인" 알림 + 조건 안내
                     │
                     └─▶ 거절 → 신청 종료
                               └─▶ [고객] ← "심사 거절" 알림 + 사유
```

### 2.5 대출 실행 단계
```
[승인 관리 시스템]
   │
   └─▶ (승인 완료) → [대출 실행 시스템]
         │
         ├─▶ (약정 안내 발송)
         │     └─▶ [고객] ← SMS/앱 푸시/이메일
         │
         └─▶ [고객]
               │
               ├─▶ (모바일에서 약정 내용 확인)
               │     ├─ 대출 금액
               │     ├─ 금리
               │     ├─ 상환 방법
               │     └─ 약정 조건
               │
               ├─▶ (전자 서명 수행)
               │     ├─ 공동인증서
               │     └─ 간편인증
               │
               └─▶ (약정 완료) → [대출 실행 시스템]
                     │
                     ├─▶ (입금 계좌 실명 확인)
                     │     └─▶ [금융결제원 API] → 계좌 인증
                     │
                     ├─▶ (대출 실행 처리)
                     │     └─ 자금 이체 실행
                     │
                     ├─▶ (대출 원장 생성)
                     │     └─▶ [코어뱅킹 시스템 (CBS)]
                     │           └─ MQ 비동기 연동
                     │
                     └─▶ (실행 완료 통보)
                           └─▶ [고객] ← 실행 완료 알림 + 전자 약정서
```

### 2.6 상환 관리 단계
```
[대출 실행 시스템]
   │
   └─▶ (대출 실행 완료) → [상환 관리 시스템]
         │
         ├─▶ (상환 일정 등록)
         │     ├─ 매월 상환일 설정
         │     ├─ 상환 금액 계산
         │     └─ 자동이체 등록
         │
         ├─▶ (정기 상환 프로세스)
         │     │
         │     ├─▶ 상환일 3일 전
         │     │     └─▶ [고객] ← 상환 예정 알림 (SMS/앱)
         │     │
         │     ├─▶ 상환일 당일
         │     │     └─▶ 자동이체 실행
         │     │           ├─ 원금 차감
         │     │           ├─ 이자 납입
         │     │           └─ 잔액 업데이트
         │     │
         │     └─▶ 상환 완료
         │           └─▶ [고객] ← 상환 완료 알림
         │
         ├─▶ (연체 관리 프로세스)
         │     │
         │     ├─▶ 연체 1일차
         │     │     └─▶ [고객] ← SMS 자동 발송
         │     │
         │     ├─▶ 연체 7일차
         │     │     ├─▶ [여신관리팀] → 전화 독촉
         │     │     └─ 연체 이자 부과
         │     │
         │     ├─▶ 연체 30일차
         │     │     └─▶ [고객] ← 신용정보 등록 예고 알림
         │     │
         │     └─▶ 연체 90일차
         │           └─▶ [여신관리팀] → 법적 조치 진행
         │
         └─▶ (중도 상환 프로세스)
               │
               └─▶ [고객]
                     │
                     ├─▶ (중도 상환 요청)
                     │     └─▶ [상환 관리 시스템]
                     │           └─ 중도상환 수수료 자동 계산
                     │
                     ├─▶ (수수료 확인 및 동의)
                     │
                     └─▶ (중도 상환 실행)
                           ├─ 잔액 전액 상환
                           ├─ 대출 종료 처리
                           └─▶ [고객] ← 상환 완료 알림
```
